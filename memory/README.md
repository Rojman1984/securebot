# SecureBot Memory System

Personal memory files are gitignored for privacy.
Copy templates to get started:
```bash
cp memory/templates/soul.template.md memory/soul.md
cp memory/templates/user.template.md memory/user.md
cp memory/templates/session.template.md memory/session.md
cp memory/templates/tasks.template.json memory/tasks.json
```

Then run the setup wizard:
```bash
./services/scripts/memory_setup.sh
```
